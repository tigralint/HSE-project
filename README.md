Проект «НОЧНОЙ СТРАННИК»
Интерактивный веб-квест и персональный сайт-портфолио, созданный на основе авторских графических дизайнов. Проект представляет собой погружение в историю «скитальца» через исследование цифровых артефактов.

🌌 О проекте
Сайт реализован как заброшенный терминал или личный архив персонажа. Основная задача заключалась в том, чтобы перенести визуальные идеи дизайнера в живой, интерактивный веб-интерфейс с сохранением уникальной атмосферы (пиксель-арт, нуар, эстетика старых интерфейсов).

📂 Структура и разделы
Главная страница: Входная точка с интерактивным персонажем, который реагирует на движение курсора.
Дневник (Terminal Style): Лог-файлы записей странника, оформленные в стиле командной строки с неоновой подсветкой.
Карта: Интерактивный артефакт мира «Smokey Town» с эффектом разворачивания.
Фотопленка: Галерея в виде горизонтальной кинопленки с перфорацией и возможностью детального просмотра снимков через Lightbox.
Рация (Форма связи): Тематическая анкета «Давай дружить» для взаимодействия с посетителем.
Учебные задачи: Набор функциональных страниц (паспорт с транслитерацией, алгоритмический калькулятор).

🛠 Технологический стек
HTML5: Семантическая верстка, формы, таблицы.

CSS3:
Сложные анимации (flicker, glitch, bobbing, unfold).
Кастомные шрифты и стилизация под ретро-интерфейсы.
Полная адаптивность под мобильные устройства (Media Queries).

JavaScript (Native):
Интерактивная логика для персонажа на главной.
Собственная реализация системы Lightbox для просмотра изображений.
Динамическое изменение контента (транслитерация) и решение алгоритмических задач.

✨ Особенности реализации
Интерактив: Персонаж на главной странице «следит» взглядом за мышью, создавая эффект присутствия.
Атмосферность: Использование глитч-эффектов, мерцания текста и тематических фонов для полного погружения в дизайн.
Адаптивность: Интерфейс перестраивается под разные экраны, сохраняя читаемость «терминала» и удобство навигации.
Проект выполнен в рамках технической помощи по реализации авторских дизайнов Елизаветы Харченко.
